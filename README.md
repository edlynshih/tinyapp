## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session
- body-parser

## Dev Dpendencies

- Mocha
- Chai
- Nodemon

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
- Go to `http://localhost:8080/` in your browser
- Enjoy the features!
